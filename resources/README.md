# Resources

This directory contains shared resources, utilities, and reference materials used across multiple courses and projects.

## Contents

- **Utilities**: Reusable code, helper functions, and common tools
- **References**: Papers, tutorials, and learning materials
- **Templates**: Project templates and boilerplate code
- **Datasets**: Links or small datasets used across multiple projects

## Organization

```
resources/
├── utils/                 # Shared utility functions
├── templates/             # Project templates
├── datasets/              # Shared datasets or links
└── references/            # Papers, articles, and learning materials
```
