# Penn State Master of AI Coursework

A repository for all my Master of AI code at Penn State University.

## Repository Structure

This repository is organized by course and project to maintain clear separation of coursework and facilitate easy navigation.

```
penn-state-classes/
├── courses/           # Course-specific work
│   ├── course-1/      # Individual course directories
│   ├── course-2/
│   └── ...
├── projects/          # Major projects and capstone work
│   ├── project-1/
│   └── ...
├── resources/         # Shared resources, utilities, and references
└── README.md          # This file
```

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip or conda for package management
- Git for version control

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/hubbertj/penn-state-classes.git
   cd penn-state-classes
   ```

2. Create a virtual environment (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install dependencies for specific courses as needed (check individual course directories for requirements).

## Course Directory Guidelines

Each course directory should contain:
- `README.md` - Course description and assignment overview
- `requirements.txt` - Python dependencies for the course
- Assignment/project subdirectories with clear naming
- Any course-specific resources or notes

## Projects

Major projects and capstone work are stored in the `projects/` directory, each with its own README and documentation.
